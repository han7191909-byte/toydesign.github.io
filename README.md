# toydesign.github.io
本作は、ハンドジェスチャー認識によって操作される体験型のゲーム作品である。 プレイヤーの指先は一つひとつの「星」となり、身体の動きを通して、多様な幾何学的形態を生み出していく。 さらに次の段階では、手そのものが空間に描くための「筆」となり、単純な形から想像を広げながら、自由に図形を描き、物語を紡いでいくことができる。 現代のデジタル空間において、プレイヤーがただ手を伸ばすだけで直感的に形と関わり、そこから想像力を喚起し、創作する楽しさを感じられる————そのような体験を生み出したいと私は思う。

本网站是一件通过手势识别进行操作的体验型互动游戏。 在游戏中，玩家的手指尖变成了一颗星星，通过身体动作就可以创造出来各种几何图形 在随后的阶段中，手本身成为在空间中描绘的「笔」，玩家可以从简单的形状出发，不断延展想象，自由地绘制图形，并逐步编织属于自己的故事。 作者希望创造这样一种体验： 屏幕前的我们只需伸出双手，就可以直观的和各种形状产生关系，并从中得到一些想象和创作的乐趣。
